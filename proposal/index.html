نظرة عامة

اوروك موتورز هو تطبيق اشتراكات مبتكر يُتيح لأصحاب السيارات الحصول على تغطية شاملة للأضرار الخارجية كالأبواب والمصدّات والهيكل، مع إمكانية الإبلاغ عن الحوادث وتحديد مواعيد الإصلاح بكل سهولة. يدعم التطبيق ثلاثة مستويات من الاشتراك لتناسب احتياجات مختلف المستخدمين، ويعمل على Android وiOS فضلاً عن الويب من قاعدة كود Flutter واحدة.

المزايا الرئيسية

اشتراكات متعددة المستويات

ثلاث خطط (ستاندرد / مشترك / VIP) بأسعار تبدأ من 25,000 دينار شهرياً

تسجيل السيارة

إدخال الماركة والموديل والسنة واللون ورقم اللوحة مع خيارات سريعة

الإبلاغ عن الحوادث

نموذج ثلاثي الخطوات: صور الضرر، التفاصيل، وبيانات الطرف الآخر

تحديد المواعيد

حجز وإدارة مواعيد الإصلاح مع إمكانية طلب تغيير الموعد

إدارة الدفعات

متابعة سجل الدفعات الشهرية عبر ماستركارد أو الدفع الإلكتروني

التحقق بالهوية والـ OTP

تسجيل آمن مع رفع صور الهوية الوطنية والتحقق برمز OTP

الإشعارات الفورية

تنبيهات بحالة الطلبات والمواعيد والدفعات المستحقة

لوحة تحكم ويب للإدارة

واجهة للموظفين (مدير / موظف / ورشة) لمتابعة الطلبات والمشتركين

دور الإدارة (Administration)

مسؤول متخصص يملك صلاحية إنشاء حسابات الموظفين والموافقة على طلبات الانضمام أو رفضها

طلبات تعديل بيانات السيارة

المشترك يقدّم طلب تعديل للإدارة (اللون، اللوحة، الموديل…) وتُراجعه الإدارة وتوافق أو ترفض

مسار المستخدم

رحلة المستخدم عبر التطبيق تسير وفق التسلسل الآتي:

1

شاشة البداية (Splash)

شاشة تمهيدية متحركة تعرض شعار اوروك موتورز لمدة ثانيتين

2

شاشة الترحيب (Onboarding)

ثلاث شرائح تعريفية بمميزات التطبيق مع أزرار التالي والتخطي

3

تسجيل الدخول / إنشاء حساب

مصادقة برقم الهاتف وكلمة المرور، أو إنشاء حساب جديد مع التحقق بـ OTP

4

رفع الهوية وانتظار الموافقة

يرفع المستخدم صورتَي هويته الوطنية ثم ينتظر موافقة الإدارة

5

اختيار خطة الاشتراك

عرض الخطط الثلاث مع المزايا والأسعار واختيار المناسب والتأكيد

6

تسجيل السيارة

إدخال بيانات السيارة: الماركة، الموديل، السنة، اللون، رقم اللوحة

7

الشاشة الرئيسية

ملخص الاشتراك، الإجراءات السريعة، الدفعات المستحقة، الحوادث الأخيرة

8

الوظائف الرئيسية عبر شريط التنقل

الإبلاغ عن الحوادث • المواعيد • الدفع • الملف الشخصي • الإشعارات

المكدس التقني

تم اختيار هذه التقنيات لضمان أداء عالٍ وقابلية التوسع واستخدام مكتبات مجربة في بيئات الإنتاج.

📱 إطار العمل

Flutter (Dart) — تطوير متعدد المنصات: Android، iOS، Web من قاعدة كود واحدة

🧭 التنقل

go_router — تنقل تصريحي مع ShellRoute للشريط السفلي ودعم المسارات المتداخلة

🔄 إدارة الحالة

Provider (ChangeNotifier) — خفيف وموثوق، جاهز للتحول إلى Riverpod أو Bloc

🎨 الخط والثيم

Poppins (Google Fonts)، ثيم مخصص بلون #1A3A8F، دعم كامل للعربية RTL

📷 رفع الصور

image_picker — التقاط صور الحوادث والهوية من الكاميرا أو المعرض

🔢 رمز OTP

pin_code_fields — حقل إدخال رمز التحقق 6 خانات مع دعم الإكمال التلقائي

📅 التاريخ والعملة

intl — تنسيق التواريخ والعملة (دينار عراقي) مع دعم اللغة العربية

✨ الرسوم المتحركة

flutter_animate — تأثيرات الدخول والتلاشي في الشاشات الرئيسية

🌐 الترجمة

flutter_localizations — دعم تعدد اللغات مع GlobalMaterialLocalizations للعربية

🔧 الخدمات

MockService — جميع نداءات API مُحاكاة حالياً، جاهزة للتكامل مع الباك إند

💾 التخزين المحلي

shared_preferences — حفظ بيانات الجلسة وإعدادات المستخدم محلياً

🖼️ الصور المخزّنة

cached_network_image — تحميل وتخزين مؤقت لصور المستخدمين والسيارات
